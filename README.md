# package-compiler-library-with-preferences

Minimum working example for testing how PackageCompiler.jl-generated libraries interact with
Preferences.jl.

## Usage

Build library:
```shell
make lib
```

Build executable:
```shell
make mwe
```

Run MWE:
```shell
./mwe
```
